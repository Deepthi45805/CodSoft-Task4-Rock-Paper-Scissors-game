# Rock Paper Scissors Game (Python)

## Description
This project is a simple Rock-Paper-Scissors game developed using Python. The user plays against the computer by choosing rock, paper, or scissors. The computer randomly selects its choice, and the program determines the winner based on the game rules.

## Features
- User can choose rock, paper, or scissors
- Computer randomly selects its choice
- Game determines the winner automatically
- Displays result (win, lose, or tie)
- Option to play multiple rounds

## Technologies Used
- Python
- random module

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run the Python file.

python rock_paper_scissors.py

4. Enter your choice (rock, paper, or scissors) and see the result.

## Game Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## Example Output
Enter your choice: rock  
Computer chose: scissors  
Result: You win!
